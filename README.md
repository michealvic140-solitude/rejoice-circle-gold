# 💰 Rejoice Trust Ajo Platform 💎

A luxury, high-fidelity fintech solution designed for **Rotating Savings and Credit Associations (ROSCA)**. This platform modernizes the traditional "Ajo" or "Esusu" system with automated logic, secure tracking, and a premium user interface.

## 🚀 Overview
The **Rejoice Trust Ajo** platform provides a secure environment for community-based savings. It removes the stress of manual record-keeping by automating payouts, tracking contributions, and managing user cycles in real-time.

## ✨ Key Features
* **Luxury Dashboard:** A sleek, dark-themed UI designed for a premium user experience.
* **Automated Payout Engine:** Intelligent logic to manage rotation cycles and recipient payouts.
* **Role-Based Access:** * **Admin:** Complete control over cycles, user approvals, and financial logs.
    * **User:** Personal contribution tracking and payout scheduling.
* **Real-time Ledger:** Integrated with Supabase for secure, instant transaction updates.
* **Mobile-First Design:** Fully optimized for Android and iOS mobile browsers.

## 🛠️ Tech Stack
* **Frontend:** React / Next.js with Tailwind CSS
* **Backend:** Node.js 
* **Database:** Supabase (PostgreSQL)
* **Version Control:** GitHub
* **Development Environment:** Replit

## 📂 Project Structure
* `/src/components` - UI components (Cards, Tables, Modals)
* `/src/logic` - Payout algorithms and rotation scheduling
* `/supabase` - Database schemas and security policies

## 🛠️ Setup & Installation
1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/michealvic140-solitude/rejoice-trust-ajo-platform1.4.git](https://github.com/michealvic140-solitude/rejoice-trust-ajo-platform1.4.git)
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Environment Variables:**
    Create a `.env` file and add your Supabase URL and API keys.
4.  **Run Development Server:**
    ```bash
    npm run dev
    ```
